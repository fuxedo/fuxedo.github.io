Fuxedo

"Fuxedo is a one-piece tuxedo that combines the elegance of a tuxedo and the convenience of a jumpsuit."

Fuxedo is a middleware for C and C++ programming languages written in C++14 and uses UNIX message queues for inter-process communication. It implements X/Open XATMI specification and some Oracle Tuxedo extensions of the API.

Current features

  • Standards based APIs - The Open Group XATMI
  • Typed buffers
    • STRING - C-style null-terminated
    • CARRAY - binary blobs
    • FML32 - self-describing fielded buffer like binary XML or JSON
